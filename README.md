# Reproductive Health Application

A full-stack mobile application designed to support users in tracking, understanding, and managing their reproductive health across different life stages.

The system includes a React Native (Expo) mobile application, a Node.js/Express backend API, and a MongoDB Atlas cloud database with secure authentication, personalised insights, and educational content.

---

## Features

- User authentication with JWT
- Secure password encryption using bcrypt
- Health logging (cycle tracking, medication, checkups, lifestyle data)
- Personalised statistics and insights
- Educational content (Articles & Supplements)
- Smart reminders and notifications
- Cloud database integration (MongoDB Atlas)

---

## Technologies Used

- React Native (Expo)
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication
- bcrypt
- Wikipedia API
- Git & GitHub

## How to Run the Project
### 1. Clone the repository

```bash
git clone https://github.com/SylviaMartin1/repro-health-app-2026.git
```

### 2. Navigate into the project's root folder
```bash
cd repro-health-app-2026
```

### 3. Move into the backend folder
```bash
cd evergreen-api
```

### 4. Start the backend server
```bash
npm run dev
```

### 5. Navigate into the frontend folder
```bash
cd evergreen-mobile
```

### 6. Start the Expo development folder
```bash
npm run start
```
